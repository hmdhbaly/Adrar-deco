# Adrar Deco

Static React website for Adrar Deco in Nouakchott: interior design, custom furniture, decoration, portfolio gallery, and WhatsApp-first quote flow.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Edit Content

- Services: `src/data/services.js`
- Portfolio projects: `src/data/projects.js`
- WhatsApp number and default message: `src/utils/whatsapp.js`


## Deploy

Push the repository to GitHub, import it into Vercel, and use the default Vite settings:

- Build command: `npm run build`
- Output directory: `dist`
